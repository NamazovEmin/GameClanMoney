Добрый день.

Для запуска программы перейдите в файл  game -> dbPostgres -> DataBaseConnection
И введите свои данные для подключения к базе данных Postgres.

Затем в этой же папке dbPostgres запустите все поля init_postgressql.sql
Эти скрипты создадут все нужные таблицы и заполнят их.

После этого запускаем клас Main
